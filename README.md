# SimpleFtp

A lightweight, reusable C# FTP library with async support, designed to be easy to integrate into your projects.  

It provides methods to set credentials, list directories, and test connections, and includes a minimal example demonstrating usage.

---

## Features

- Set FTP credentials with `FtpClient.SetCredentials`
- List directories asynchronously with `FtpClient.GetDirectoriesAsync`
- Test connection to an FTP server with `FtpClient.TestConnectionAsync`
- Fully async, resource-safe with `using` blocks
